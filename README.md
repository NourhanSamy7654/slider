# Image Slider Project

A simple front-end slider project built using **HTML, CSS, and JavaScript**.

## 📌 Features

- Display images in a slider
- Next & Previous navigation buttons
- Smooth transition effect

## 🛠️ Technologies Used

- CSS
- JavaScript
